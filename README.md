# ☕ Coffee Shop Landing Page

A modern and responsive Coffee Shop Landing Page built using HTML, Tailwind CSS, and JavaScript. The project features a premium user interface, smooth animations, and an interactive coffee showcase experience.

## 🚀 Features

- Responsive design for mobile, tablet, and desktop
- Modern and premium coffee shop UI
- Interactive coffee slider
- Smooth animations and transitions
- Clean and organized code structure
- Fast and lightweight performance
- Mobile-first development approach

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
